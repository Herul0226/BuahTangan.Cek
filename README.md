# BuahTangan.Cek

Website katalog produk handmade crochet berkualitas tinggi.

## 📁 Struktur File

```
crochet-static/
├── index.html      # File HTML utama
├── style.css       # Styling CSS
├── script.js       # JavaScript interaktif
├── README.md       # Dokumentasi
└── vercel.json     # Konfigurasi Vercel
```

## 🚀 Cara Deploy ke Vercel

### Opsi 1: Upload Manual
1. Buka https://vercel.com
2. Klik "Add New Project"
3. Pilih "Import Project"
4. Upload folder ini atau connect ke GitHub
5. Klik "Deploy"

### Opsi 2: Menggunakan Vercel CLI
```bash
npm i -g vercel
cd crochet-static
vercel
```

## 🎨 Customization

### Warna
Edit di `style.css` bagian `:root`:
```css
:root {
    --primary: #001F5C;      /* Navy Blue */
    --secondary: #4A90E2;    /* Sky Blue */
    --accent: #2D8659;       /* Green */
    --red: #E63946;          /* Red */
    --pink: #F4A6C1;         /* Pink */
}
```

### Font
Fonts sudah di-import dari Google Fonts:
- Display: Poppins
- Body: Inter

### Produk
Edit di `index.html` bagian "KATALOG PAGE" untuk menambah/mengubah produk
